# MusaCare (Doctor-Side Android App)

MusaCare is an offline-first psychologist support app built with a 100% free/open-source Android stack.

## Stack
- Kotlin
- Jetpack Compose
- MVVM
- Room
- Hilt
- Coroutines + Flow

## Features in this version
- PIN-based local app lock
- Patient management (add/archive/list)
- Session logs
- Notes and quick local search-ready data model
- Mood tracking
- Psychological tests catalog + assignment
- Reminder creation
- Offline local-first storage
